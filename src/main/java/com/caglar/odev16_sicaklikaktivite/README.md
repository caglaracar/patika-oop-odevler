# Ödev 16 - Sıcaklığa Göre Aktivite Öneri

## Problem
- Sıcaklık < 5 → Kayak
- 5 ≤ Sıcaklık < 15 → Sinema
- 15 ≤ Sıcaklık < 25 → Piknik
- Sıcaklık ≥ 25 → Yüzme

## Ödev
Aynı problemi **farklı yöntemlerle** çöz.

## Yaklaşımlar
1. **if-else if zinciri**
2. **Ternary operator** (iç içe)
3. **Ayrı if bloklar** (`&&` ile aralık kontrolü)
4. **switch-case** (aralığı önce bir sayıya çevirip)

## Çalıştırma
`Main.java` çalıştırılır, sıcaklık girilir, 4 yöntemin çıktısı yan yana görünür.
