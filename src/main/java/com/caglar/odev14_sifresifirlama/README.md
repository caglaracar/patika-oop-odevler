# Ödev 14 - Şifre Sıfırlama

## Problem
- Kullanıcı şifreyi yanlış girerse şifresini sıfırlamak isteyip istemediği sorulur.
- Sıfırlamak isterse yeni girilen şifre, unutulan şifre ile **aynı olmamalıdır**.
- Aynı ise: "Şifre oluşturulamadı, lütfen başka şifre giriniz."
- Farklı ise: "Şifre oluşturuldu"

## Çalıştırma
`Main.java` çalıştırılır. Doğru şifre kodda `1234` olarak tanımlı.
