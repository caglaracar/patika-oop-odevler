# Ödev 10 - Dairenin Alanı ve Çevresi + Daire Dilimi

## Problem
Yarıçapı kullanıcıdan al, dairenin alanı ve çevresini hesapla.

## Ödev
Merkez açısı α olan daire diliminin alanını bul.

## Formüller (π = 3.14)
- Alan = π * r * r
- Çevre = 2 * π * r
- Daire Dilimi Alanı = (π * r * r * α) / 360

## Çalıştırma
`Main.java` çalıştırılır, yarıçap ve merkez açısı girilir.
