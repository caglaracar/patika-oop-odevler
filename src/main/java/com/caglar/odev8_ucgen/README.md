# Ödev 8 - Üçgen Alanı (Heron Formülü)

## Problem
Asıl problem: Dik üçgende hipotenüs bulmak.
Ödev: Üç kenarı bilinen bir üçgenin alanını Heron formülü ile hesapla.

## Formül
- u = (a + b + c) / 2
- Alan² = u * (u - a) * (u - b) * (u - c)
- Alan = √(Alan²)

## Çalıştırma
`Main.java` çalıştırılır, üç kenar girilir.
