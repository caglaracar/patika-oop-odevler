# Ödev 6 - Not Ortalaması Hesaplayan Program

## Problem
6 dersin (Matematik, Fizik, Kimya, Türkçe, Tarih, Müzik) notlarını kullanıcıdan al, ortalamayı hesapla ve ekrana yazdır.

## Ödev
- Ortalama > 60 → "Sınıfı Geçti"
- Ortalama <= 60 → "Sınıfta Kaldı"
- **Ternary operator** ile kontrol yapılır.

## Çalıştırma
`Main.java` çalıştırılır, notlar sırayla girilir.
