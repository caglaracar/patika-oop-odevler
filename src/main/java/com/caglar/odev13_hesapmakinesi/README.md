# Ödev 13 - Hesap Makinesi (switch-case)

## Problem
İki sayı ve bir işlem türü (`+`, `-`, `*`, `/`) alarak sonucu hesaplayan program.
`switch-case` yapısı kullanılır.

## Özellikler
- Sıfıra bölme kontrolü yapılır.
- Geçersiz işlem için default case hata mesajı verir.

## Çalıştırma
`Main.java` çalıştırılır, sayılar ve işlem girilir.
