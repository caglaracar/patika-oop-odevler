# Ödev 11 - Vücut Kitle İndeksi Hesaplama

## Problem
Kullanıcıdan boy (metre) ve kilo (kg) al, VKİ hesapla.

## Formül
`VKİ = kilo / (boy * boy)`

## Örnek Çıktı
```
Lütfen boyunuzu (metre cinsinde) giriniz: 1.72
Lütfen kilonuzu giriniz: 105
Vücut Kitle İndeksiniz: 35.49215792320173
```
