# Ödev 21 - Artık Yıl Kontrolü

## Problem
Kullanıcının girdiği yılın artık yıl olup olmadığını belirle.

## Kurallar
- 4'e tam bölünen yıllar artık yıldır.
- Ancak 100'e tam bölünenler artık yıl DEĞİLDİR.
- İstisna: 400'e tam bölünenler artık yıldır.

Örnekler:
- 2000 → artık yıl (400'e bölünür)
- 1900 → artık yıl değil (100'e bölünür ama 400'e bölünmez)
- 2020 → artık yıl (4'e bölünür, 100'e bölünmez)

## Çalıştırma
`Main.java` çalıştırılır.
