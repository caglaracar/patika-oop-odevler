# Ödev 9 - Taksimetre Programı

## Problem
Gidilen mesafeye (km) göre taksimetre tutarını hesapla.

## Kurallar
- KM başına: 2.20 TL
- Açılış ücreti: 10 TL
- Minimum ödenecek tutar: 20 TL (hesap 20 TL'nin altındaysa 20 TL alınır)

## Formül
`tutar = açılışÜcreti + (mesafe * kmFiyatı)`
`tutar < 20` ise tutar = 20

## Çalıştırma
`Main.java` çalıştırılır, mesafe girilir.
